# parrotsay-api

## Install

```
npm install --save parrotsay-api
```
## Compile

Using node@v5

```
npm install -g gulp@3.9.0
gulp
```
## Usage

```js
// example.js
const parrotSay = require('parrotsay-api')

parrotSay('yaaay')
  .then(console.log)
  .catch(console.error)
```
